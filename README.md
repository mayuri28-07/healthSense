HealthSense is a simple and responsive web page designed to promote health awareness and provide basic wellness information. This project is built using HTML and CSS, focusing on clean design, readability, and user-friendly navigation.

The goal of this project is to demonstrate front-end development skills while creating a meaningful and informative health-related webpage.
